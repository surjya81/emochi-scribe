# 📥 Emochi Scribe — Chrome Extension

A powerful, privacy-first Chrome extension to seamlessly download your Emochi.com roleplay conversations and chats. It  passively or actively records your chat as you scroll and exports it instantly.

---

## ✨ Features

- **Zero API Errors:** Uses passive interception. It safely reads the data your browser already receives without triggering Cloudflare blocks.
- **🚀 Auto-Scroller:** Got a chat with 1,000+ messages? Click "Start Auto-Scroll" and watch the extension pull ~50 messages per second until it reaches the very top.
- **Smart Filtering:** Automatically ignores discarded alternate swipes and only exports your active timeline.
- **Multiple Formats:** Download as `.TXT` (for simple reading), `.MD` (for Obsidian/Notion), or `.JSON` (for developers and backups).

---

## 🛠️ Installation

1. Open Google Chrome and type `chrome://extensions/` in the URL bar.
2. Turn on **Developer Mode** (toggle in the top-right corner).
3. Click the **"Load unpacked"** button.
4. Select the folder containing this extension.
5. The extension icon (a purple download arrow) will appear in your browser toolbar!

---

## 📖 How to Use

### Method 1: The Floating Panel (Easiest)
1. Go to any Emochi chat page (e.g., `emochi.com/character/.../chat`).
2. You will see a purple **📥** button floating in the bottom-right corner. Click it.
3. **To load history:** Just scroll up in your chat normally, OR click **🚀 Start Auto-Scroll** to let the extension do the work.
4. Watch the message count tick up!
5. Select your desired format (TXT, MD, or JSON) and click **⬇ Download**.

### Method 2: The Toolbar Popup
1. While chatting on Emochi, click the extension icon in your Chrome toolbar.
2. The popup communicates live with the webpage.
3. You can trigger the **Auto-Scroll** directly from the popup.
4. Choose your format and click Download.

---

## 💡 Tips & Tricks

- **Fresh Start:** If the extension says "Initializing..." forever, just do a hard refresh of the page (`F5` or `Ctrl+R`). The extension needs to be active when the page first loads to intercept the data.
- **Switching Chats:** The extension is smart! If you click a new character in the sidebar, the extension automatically wipes its memory and resets to 0 so your downloads never get mixed up.

---

## 🔧 Troubleshooting

**"Please refresh the chat page to enable interception"**
You opened a chat before the extension was fully loaded. Simply refresh the browser tab.

**Downloads aren't saving?**
Check Chrome Settings → Downloads to ensure Chrome isn't blocking files generated by extensions.
