# 📥 Emochi Scribe — Chat Downloader & Exporter

Looking for a way to export your Emochi chat? **Emochi Scribe** is a free, privacy-first Chrome extension that allows you to easily download, save, and backup your Emochi.com / FlowGPT AI roleplay conversations. 

Stop copy-pasting your AI stories message-by-message. This tool passively records your chat as you scroll and exports the entire history instantly.

---

## ✨ Key Features

- **Save AI Stories Instantly:** Export your entire chat history as `.TXT` (for reading), `.MD` (for Obsidian/Notion), or `.JSON` (for developers).
- **🚀 Auto-Scroller:** Have a massive 1,000+ message roleplay? Click "Start Auto-Scroll" and the extension will automatically scroll to the very top, capturing ~50 messages per second.
- **Smart Filtering:** Automatically ignores discarded alternate swipes and only exports your active, final timeline.
- **Zero Account Bans:** Uses passive network interception. It safely reads the data your browser already receives, meaning it will not trigger Cloudflare blocks or API bans.

## 🛠️ How to Install the Extension

Since this tool is currently in beta, you can install it directly from this repository:

1. Click the green **"Code"** button above and select **"Download ZIP"**.
2. Extract the ZIP file to a folder on your computer.
3. Open Google Chrome and type `chrome://extensions/` in the URL bar.
4. Turn on **Developer Mode** (toggle in the top-right corner).
5. Click the **"Load unpacked"** button.
6. Select the folder you just extracted.
7. The Emochi Scribe icon (a purple download arrow) will appear in your browser toolbar!

## 📖 How to Download Your Emochi Chat

### Method 1: The Floating Panel (Recommended)
1. Go to any Emochi chat page (e.g., `emochi.com/character/.../chat`).
2. You will see a purple **📥** button floating in the bottom-right corner. Click it.
3. **To load your history:** Scroll up in your chat normally, OR click **🚀 Start Auto-Scroll** to let the extension do the work.
4. Watch the message count tick up!
5. Select your desired format (TXT, MD, or JSON) and click **⬇ Download**.

### Method 2: The Toolbar Popup
1. While chatting on Emochi, click the extension icon in your Chrome toolbar.
2. The popup communicates live with the webpage.
3. You can trigger the **Auto-Scroll** directly from the popup.
4. Choose your format and click Download.

## 💡 Troubleshooting & FAQs

**"Please refresh the chat page to enable interception"**
You opened a chat before the extension was fully loaded. Simply refresh the browser tab (`F5`) so the extension can hook into the page.

**How do I backup a different character?**
The extension is smart. If you click a new character in the sidebar, the extension automatically wipes its memory and resets to 0 so your downloads never get mixed up.

**Downloads aren't saving?**
Check your `Chrome Settings → Downloads` to ensure Chrome isn't blocking files generated by extensions.
